# 最简单的部署方式：Vercel GitHub 集成

## 只需 3 步：

### 第 1 步：访问 Vercel
打开 https://vercel.com/new

### 第 2 步：导入 GitHub 仓库
1. 点击 **Import Git Repository**
2. 找到 `cosmetic-articles-search`
3. 点击 **Import**

### 第 3 步：部署
1. 什么都不用改，直接点击 **Deploy**
2. 等待 1-2 分钟
3. 完成！Vercel 会给你域名

---

## 后续更新（自动）

配置完成后，以后你只需要：
1. 修改代码
2. `git push` 到 GitHub
3. Vercel 自动重新部署

不需要再手动操作！

---

## 绑定自定义域名（可选）

部署完成后：
1. 进入 Vercel Dashboard → 你的项目 → Settings → Domains
2. 添加你的域名
3. 在域名服务商添加 DNS 记录
